## Editable Models In No Time

```php
$form->image('profile_image')
    ->maxFiles(1)
    ->title('Profile Image')
    ->crop(3 / 4)
    ->width(4);

$form->group(function ($col) {
    $col->input('name')
        ->title('Name');

    $col->relation('department')
        ->title('Department');
})->width(2 / 3);
```
